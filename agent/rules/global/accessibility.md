# Global Accessibility Rules (EDS blocks)

Minimum expectations for generated blocks:

- **Interactive controls**: use real `button`/`a` elements; ensure keyboard access.
- **Focus**: do not remove focus outlines; ensure visible focus state.
- **Images**: meaningful `alt` for informative images; empty alt for decorative.
- **Headings**: use appropriate heading levels; avoid skipping without reason.
- **ARIA**: add only when needed; keep roles/attributes correct (especially for tabs).
- **Color contrast**: flag likely contrast issues; prefer existing token colors.
